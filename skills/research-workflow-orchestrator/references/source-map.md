## Adaptation Notes

- K-Dense material is used as workflow guidance, not installed wholesale.
- pyzotero is treated as a Web API fallback, not the primary Zotero path.
- Hermes material is used for iterative paper workflow concepts, not platform-specific execution.
- nature-skills is MIT licensed and used as a reference layer for high-standard figure, prose-polish, and paper-to-PPT workflows.
- DeepLearningDrawingTemplate is Apache-2.0 licensed in the local clone, but it contains README previews and commercial links rather than editable PPT source files. It is used only as inspiration for network-architecture visual grammar, not copied as assets.
- The workflow is designed for Windows + Codex and avoids hard-coded API credentials.
