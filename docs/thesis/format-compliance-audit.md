## 3. 10B-1 审计门槛

进入 10B-1 后，必须优先处理：

1. 将章节草稿中的引用占位符替换为顺序编码引用。
2. 建立图表清单，确定每张图和每张表所在章节。
3. 为每张图和每张表生成学校模板风格题注。
4. 修正摘要关键词数量。
5. 将“第 7 章 总结与展望”的正文内容映射为正式“结  论”。

## 4. 11 阶段 DOCX 生产审计门槛

DOCX 初稿生成后，必须检查：

1. 页边距、页眉页脚、页码分节是否正确。
2. 目录是否自动生成且只显示到三级。
3. 图表题注和交叉引用是否更新成功。
4. 表格是否为三线表或符合模板线宽。
5. 所有数字、英文字母和英文参考文献是否为 Times New Roman。
6. PDF 导出后是否存在图表错位、题注分页、公式截断、页眉错乱。
7. 全文字数是否不少于 3 万字。

## 5. 12 阶段兼容性审计门槛

Windows / WPS / Word 兼容性检查必须确认：

1. Mac 生成的 DOCX 在 Windows Word 和 WPS 中打开后页码不漂移。
2. 奇偶页页眉仍然正确。
3. 目录、图表题注、交叉引用域更新后无错误。
4. PDF 导出与 Mac PDF 在分页和字体上基本一致。
5. 提交版 PDF 无图题丢失、表格跨页异常、中文字体乱码。

## 6. 当前结论

10B-0 的学校模板格式锁定已经完成。后续 10B-1 可以开始处理正式引用、图表编号、图注和正文交叉引用，但所有操作必须以 `school-template-style-map.md` 为硬约束。
