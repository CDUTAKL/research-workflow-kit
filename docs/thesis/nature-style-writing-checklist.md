# Nature-Style Writing Checklist

## Purpose

Use this file as a high-standard writing and review pass after evidence, citations, and figure records are stable.

This checklist adapts useful `nature-writing`, `nature-polishing`, and reviewer-style checks into a graduation-thesis workflow. It must not be used to invent novelty, citations, or results.

## Writing Checks

| Area | Check |
|---|---|
| Introduction gap | moves from field problem to specific gap, then to this thesis contribution |
| Related work grouping | groups papers by concept, method, dataset, or limitation instead of only chronology |
| Methods reproducibility | data, method, implementation, metric, and environment details are findable |
| Results discipline | reports evidence and comparisons without turning every number into a claim |
| Discussion boundary | explains meaning, limitations, and failure cases without overgeneralizing |
| Citation precision | citations attach to the exact sentence or segment they support |
| Hedging | terms such as robust, significant, first, best, and generalizable match evidence strength |
| Thesis style | Chinese academic tone remains clear and formal without empty slogans |
