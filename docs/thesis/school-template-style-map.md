## 1. 锁定结论

本文最终 DOCX 排版必须优先遵循成都理工大学 20230905 专业学位硕士论文模板。公开旧版格式文件仅作为同源佐证，不覆盖 20230905 模板。

本阶段锁定两项后续不再默认摇摆的格式决策：

1. 参考文献采用顺序编码制，并按 GB/T 7714-2015 编排。
2. 当前草稿中的“第 7 章 总结与展望”在最终 DOCX 中转换为不编号的“结  论”部分；草稿文件名可保留 `chapter7-*`，但正式排版标题不写“第 7 章”。

如导师或学院另有明确要求，上述两项在进入 10B-1 前统一修改；进入 10B-1 后不再临时混用。

## 2. 页面和分节样式

| 项目 | 锁定值 | DOCX/WPS 操作要求 |
|---|---|---|
| 纸张 | A4 | 不使用 Letter。 |
| 页边距 | 上 2.54 cm；下 2.54 cm；左 3.17 cm；右 3.17 cm | 全文统一；图表按版心宽度缩放。 |
| 装订线 | 左 0 cm | 不额外增加装订线。 |
| 页眉距边界 | 1.5 cm | 页眉下方有横线。 |
| 页脚距边界 | 1.75 cm | 页码居中，不能过低。 |
| 封面、扉页、声明页 | 不编页码；不使用正文式页眉 | 作为独立前置分节处理。 |
| 中英文摘要 | 大写罗马数字页码，连续编码 | 模板写明摘要页码用大写罗马数字。 |
| 目录 | 小写罗马数字页码；目录页不录入目录 | 若 WPS 自动目录页码策略冲突，以模板为准人工核对。 |
| 正文及文后部分 | 从第 1 章开始阿拉伯数字从 1 连续编号至文末 | 包括结论、致谢、参考文献、成果、附录。 |
| 奇偶页页眉 | 奇数页为本章标题；偶数页为“成都理工大学硕士学位论文” | 需要开启“奇偶页不同”。 |

## 3. 全局字体与段落规则

| 对象 | 中文字体 | 西文/数字字体 | 字号 | 段落规则 |
|---|---|---|---|---|
| 正文 | 宋体 | Times New Roman | 小四 | 两端对齐；首行缩进 2 字符；段前 0；段后 0；1.25 倍行距。 |
| 英文字母、数字、指标缩写 | - | Times New Roman | 随上下文 | 全文强制检查。 |
| 页眉 | 宋体 | Times New Roman | 小五 | 居中；页眉线。 |
| 页码 | - | Times New Roman | 小五 | 页脚居中；页码两侧不加短横线。 |
| 脚注 | 宋体 | Times New Roman | 小五 | 左对齐；无缩进；单倍行距；每页重新编号。 |

全局注意：公式编号、图表编号、百分数、变量、指标缩写、参考文献英文条目中的数字和字母均应使用 Times New Roman。

## 4. 前置部分结构

最终 DOCX 前置部分按以下顺序装配：

1. 封面。
2. 扉页。
3. 独创性声明和学位论文使用授权书。
4. 中文题名页与中文摘要。
5. 英文题名页与英文摘要。
6. 目录。

若图表较多，可按学院要求在目录后增加图清单和表清单；当前先不默认增加，除非导师要求。

## 5. 摘要与关键词样式

| 样式角色 | 锁定格式 |
|---|---|
| 中文题名 | 另起一页；黑体/Times New Roman；小三；加粗；居中；单倍行距。 |
| 中文摘要标题 | 写作“摘  要”；宋体；小三；加粗；居中；段前 18 磅；段后 18 磅；单倍行距；大纲级别 1。 |
| 中文摘要正文 | 宋体/Times New Roman；小四；两端对齐；首行缩进 2 字符；段前段后 0；1.25 倍行距。 |
| 中文关键词 | “关键词：”宋体小四加粗；关键词宋体/Times New Roman 小四；3-5 个；全角分号“；”分隔；最后不加标点；段前段后 1 行。 |
| 英文题名 | 另起一页；Times New Roman；小三；加粗；居中；单倍行距。 |
| Abstract 标题 | Times New Roman；小三；加粗；居中；段前 18 磅；段后 18 磅；单倍行距；大纲级别 1。 |
| 英文摘要正文 | Times New Roman；小四；两端对齐；首行缩进 2 字符；段前段后 0；1.25 倍行距；英文标点后保留一个半角空格。 |
| 英文关键词 | “Keywords:” Times New Roman 小四加粗；关键词 Times New Roman 小四；3-5 个；半角分号“;”分隔；最后不加标点；段前段后 1 行。 |

10B-1 前置修正：10A-9 摘要草稿中的关键词必须压缩到 3-5 个，并保持中英文一一对应。

## 6. 目录样式

| 对象 | 锁定格式 |
|---|---|
| 标题 | “目  录”；黑体；小二；加粗；居中；段前 24 磅；段后 18 磅；单倍行距。 |
| 收录级别 | 显示到 3 级。 |
| 一级目录项 | 宋体/Times New Roman；小四；左缩进 0；页码右对齐；有前导符。 |
| 二级目录项 | 宋体/Times New Roman；小四；左缩进 2 汉字符；页码右对齐；有前导符。 |
| 三级目录项 | 宋体/Times New Roman；小四；左缩进 4 汉字符；页码右对齐；有前导符。 |
| 页码 | 小写罗马数字；目录页不录入目录。 |

目录生成后必须刷新并人工检查页码、前导符和缩进。

## 7. 正文章节样式

| 样式角色 | 格式要求 | 示例 |
|---|---|---|
| 一级标题 / 章标题 | 黑体/Times New Roman；小二；加粗；居中；段前 12 磅；段后 3 磅；单倍行距；每章另起一页；题序与标题间空一个汉字或两个半角空格。 | `第1章  绪论` |
| 二级标题 | 黑体/Times New Roman；小三；加粗；两端对齐；段前 18 磅；段后 12 磅；1.5 倍行距；题序与标题间空 1 个空格。 | `1.1 研究背景与意义` |
| 三级标题 | 黑体/Times New Roman；四号；两端对齐；段前 12 磅；段后 9 磅；1.25 倍行距；题序与标题间空 1 个空格。 | `1.1.1 标题` |
| 四级标题 | 楷体/Times New Roman；小四；加粗；两端对齐；段前 12 磅；段后 6 磅；1.25 倍行距；题序与标题间空 1 个空格。 | `1.1.1.1 标题` |
| 正文段落 | 宋体/Times New Roman；小四；两端对齐；首行缩进 2 字符；段前段后 0；1.25 倍行距。 | 正文内容 |

写作层级约束：正文优先使用三级标题以内；四级标题仅在局部复杂方法或实验描述中谨慎使用。目录只显示到三级。

## 8. 图、表、公式样式

### 8.1 图

| 项目 | 锁定要求 |
|---|---|
| 插入位置 | 位于相关说明文字之后，采用嵌入式，图与正文之间空一行。 |
| 图片内部 | 不放论文图题；不出现“图 X”；不出现工程化口语如 EXP、专家等。 |
| 图题位置 | 图下方。 |
| 图题格式 | 黑体/Times New Roman；五号；居中；段前 3 磅；段后 3 磅；单倍行距。 |
| 编号 | 按章编号，如 `图4-1`。 |
| 图名标点 | 图名中不使用标点，图名后不加标点。 |
| 图注/来源 | 宋体/Times New Roman；小五；两端对齐；首行缩进 2 字符；单倍行距；必要时置于图题上方或下方，按模板实际效果统一。 |
| 分图 | 使用 `(a)`、`(b)`、`(c)`，分图名列在各自分图下方，主图名列在所有分图下方正中。 |

### 8.2 表

| 项目 | 锁定要求 |
|---|---|
| 插入位置 | 位于相关说明文字之后，表与正文之间空一行。 |
| 表题位置 | 表上方。 |
| 表题格式 | 黑体/Times New Roman；五号；居中；段前 3 磅；段后 3 磅；单倍行距。 |
| 编号 | 按章编号，如 `表5-1`。 |
| 表名标点 | 表名中不使用标点，表名后不加标点。 |
| 表格形式 | 优先三线表；上下边框 1.5 磅，其他线 0.5 磅。 |
| 表中文字 | 宋体/Times New Roman；五号。 |
| 续表 | 跨页时在续表上方注明“续上表”，并重复表头。 |
| 表注/来源 | 宋体/Times New Roman；小五；两端对齐；首行缩进 2 字符；单倍行距。 |

### 8.3 公式

| 项目 | 锁定要求 |
|---|---|
| 位置 | 公式居中。 |
| 编号 | 按章编号，如 `(4-1)`；编号置于括号内，右端对齐。 |
| 字体 | 变量和数字使用 Times New Roman；符合数学排版时变量可按公式编辑器默认斜体。 |
| 行距 | 含公式段落必要时设置为最小值 18 磅，保证公式完全显示。 |

## 9. 文后部分样式

| 部分 | 样式要求 | 处理决策 |
|---|---|---|
| 结  论 | 另起一页；不加章号；黑体小二加粗居中；段前 12 磅；段后 3 磅；正文宋体/Times New Roman 小四，首行缩进 2 字符，1.25 倍行距。 | 由第 7 章草稿转换。 |
| 致  谢 | 另起一页；标题样式同结论；正文小四，1.25 倍行距。 | 后续单独撰写。 |
| 参考文献 | 另起一页；标题黑体小二；正文宋体/Times New Roman 五号；悬挂缩进 1 字符；段前段后 0；1.25 倍行距。 | 采用顺序编码制，GB/T 7714-2015。 |
| 在读期间发表的学术论文与研究成果 | 另起一页；标题黑体小二；正文宋体五号，1.25 倍行距，按参考文献表格式编排。 | 若无成果，按学院要求写“无”或保留模板项。 |
| 附  录 | 另起一页；标题黑体小二；附录按 A、B 排列，分节如 A.1、A.2；内容五号宋体。 | 可放补充图表索引、缩写词释义、额外实验表。 |
