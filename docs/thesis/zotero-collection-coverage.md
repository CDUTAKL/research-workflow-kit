# Zotero Collection Coverage

## Purpose

Use this file to check whether Zotero collections actually cover the thesis sections, segments, and claim evidence they are supposed to support.

This complements `zotero-screening-loop.md`: screening decides what to keep; coverage shows what each chapter can safely cite and where citation gaps remain.

## Update Rules

- Map Zotero collections to `SEC-*`, `SEG-*`, and `CLM-*`.
- A collection is not “covered” just because it contains many items; core papers must have verified citation provenance.
- Use this before writing related work, before advisor review, and before final audit.
- Keep collection names human-readable and stable.
