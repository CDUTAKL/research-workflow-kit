# Figure References

Store visual reference images here.

Use this folder for Image Gen outputs, paper-example screenshots that are legally usable as private references, hand sketches, or Figma draft previews that guide composition, palette, and visual hierarchy.

Rules:

- Reference images are not final thesis or manuscript figures.
- Do not insert Image Gen reference images directly into the thesis unless explicitly approved.
- Record each reference in `docs/thesis/figure-plan.md`.
- Check content accuracy before formal redraw.
- Redraw final figures from source-of-truth records using Figma, PPTX, SVG, TikZ, Python, or an approved project renderer.
