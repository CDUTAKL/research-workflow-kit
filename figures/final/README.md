# Final Figures

Store manuscript-ready figure exports here.

Final figures should be redrawn from source-of-truth records and audited before thesis or paper insertion.

Expected checks:

- source data or architecture source is recorded.
- visual reference, if used, is recorded separately under `figures/references/`.
- caption does not exceed evidence.
- SVG/PDF/PNG/PPTX export path is recorded in `docs/thesis/figure-plan.md`.
- unwanted Image Gen C2PA/OpenAI provenance metadata is absent from final exported assets when relevant.
