# FIG-RESULT-002 Chart Choice Note

图表目标：解释 EXP-103 中为什么“直接使用事件图专家”不是安全的全局主方法，以及为什么“事件触发式预测修正”更可防守。

数据诊断：输入为冻结测试集指标，包含全局、事件窗口、稳定窗口三个层次；模型数量较少，指标均为误差类，越低越好；改进幅度在全局较小但事件窗口更明显。

图型选择：采用多面板组合。a 面板使用相对强行为基线的全局 MAE 变化，避免绝对值过小导致差异不可读；b 面板使用分窗口 MAE 柱状对照，说明事件窗口收益与稳定窗口不退化；c 面板使用误差相对变化热力图，汇总 MAE、WAPE、RMSE 三类指标。

拒绝图型：不使用三维图、面积图或过度截断坐标轴；不使用单一全局 MAE 柱图，因为无法解释稳定窗口回退和事件窗口修正的机制收益。
