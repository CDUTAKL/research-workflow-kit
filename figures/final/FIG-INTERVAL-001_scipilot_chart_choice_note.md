# FIG-INTERVAL-001 Chart Choice Note

图表目标：说明 EXP-104 的区间校准不是单纯加宽区间，而是在事件窗口修复欠覆盖，并保持整体 WIS（加权区间评分）最优。

数据诊断：输入为冻结测试集区间指标，包含校准方式、事件/稳定状态、覆盖率、WIS 和样本量；核心比较是 90% 预测区间的覆盖率与评分。

图型选择：采用三面板。a 面板展示全局 PICP 与 90% 目标线；b 面板分事件窗口和稳定窗口展示覆盖率，突出全局校准在事件窗口严重欠覆盖；c 面板展示 WIS，说明保守分组校准在覆盖率改善的同时保持更低评分。

拒绝图型：不使用单一折线图，因为校准方式是类别变量；不使用双 y 轴，以免覆盖率和 WIS 的解释混乱；不使用箱线图，因为当前源表是汇总指标而非逐样本区间宽度分布。
