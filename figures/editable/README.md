## Usage notes

- Use `figures/final/` for paper-ready PDF/SVG/PNG outputs.
- Use this directory when further manual edits are required in WPS, PowerPoint,
  Illustrator, Inkscape, or another vector editor.
- Keep figure numbers and captions outside the images; captions should be
  managed by the thesis document.
- Avoid committing temporary WPS lock files or exported duplicate previews.
